## [v1.8] - 2025-11-26
### Added
- Added ability to ctrl+backspace, ctrl+z, ctrl+y to delete previous word, undo and redo actions respectively

## [v1.7] - 2025-11-24
### Added
- Added auto copy-to-clipboard feature with handy pop-up box to verify that copying was successful.

## [v1.6] - 2025-11-24
### Added
- Added a optimisation phase cleaning up obvious mistakes via upscaling & greyscaling. Added string-repair as sometimes 'I' was read as '1' or 'l' or '|'

## [v1.5] - 2025-11-22
### Added
- Updated UI, added colour to represent status. Added auto-scaling for large amounts of text when captured. 

## [v1.4] - 2025-11-22
### Added
- Made screen darker instead of greyer. Made captured portion still bright for easier visibility.

## [v1.3] - 2025-11-22
### Added
- Added ability to capture portion of the screen and Improved UI. 

## [v1.2] - 2025-11-22
### Added
- Added simple Tkinter GUI. Still captures the entire screen so output is messy. 

## [v1.1] - 2025-11-22
### Added
- Initial release of STT. Captures entire screen and prints text to console.
